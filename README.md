# Bank Customer Retention Analysis

## Overview

This project focuses on analyzing customer churn behavior in the banking sector using Python-based data analytics techniques. The objective of the project is to identify key factors influencing customer retention and provide business recommendations to reduce churn and improve customer lifetime value.

The analysis includes customer segmentation, cohort analysis, correlation analysis, exploratory data analysis, and churn pattern identification using real-world banking customer data.

---

## Project Objectives

- Analyze customer churn behavior
- Identify factors affecting customer retention
- Perform customer segmentation using RFM-style analysis
- Conduct cohort-based customer analysis
- Generate actionable business insights and recommendations
- Support retention strategy development through data-driven analysis

---

## Dataset Information

The dataset contains banking customer information including:

- Customer demographics
- Credit score
- Geography
- Gender
- Age
- Balance
- Number of products
- Active membership status
- Estimated salary
- Churn status

## Key Business Insights

- Inactive customers showed a significantly higher churn rate.
- Customers with only one product were more likely to leave the bank.
- Geographic location influenced customer churn behavior.
- Older customers demonstrated higher churn tendencies.
- Customers with multiple banking products had better retention rates.

---

## Recommendations

- Improve customer engagement for inactive members.
- Increase cross-selling opportunities to boost product adoption.
- Develop retention programs for high-value customers.
- Create region-specific customer retention strategies.
- Use customer behavior metrics to identify churn-risk customers early.
